# premier-pr

Mon premier projet GitHub — créé pour apprendre le flux de travail avec les Pull Requests.

## Description

Ce dépôt sert d'exercice pratique pour maîtriser les bases de Git et GitHub :
- Créer une branche
- Faire un commit
- Ouvrir une Pull Request
- Merger du code

## Prérequis

- [Git](https://git-scm.com/) installé sur votre machine
- Un compte [GitHub](https://github.com)
- [GitHub CLI](https://cli.github.com/) (optionnel, mais recommandé)

## Pour commencer

```bash
# Cloner le dépôt
git clone https://github.com/SousMac/premier-pr.git

# Créer une nouvelle branche
git checkout -b ma-branche

# Faire vos modifications, puis committer
git add .
git commit -m "Ma première contribution"

# Pousser la branche
git push origin ma-branche
```

Ensuite, ouvrez une Pull Request sur GitHub !
